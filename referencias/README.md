# Referencias

Esta pasta guarda materiais usados como base para novas apresentacoes.

## Apresentacoes

- `apresentacoes/01-historia-da-biblia.pptx`: modelo com forte uso de cores, imagens e apresentacao da Vovo Docura.
- `apresentacoes/02-versiculos-pentateuco.pptx`: modelo de versiculos-chave com boa legibilidade e imagens em quase todos os slides.
- `apresentacoes/03-versiculos-livros-historicos.pptx`: modelo inicial analisado para definir o estilo do projeto.

## Como usar referencias

- Use como orientacao de ritmo, estrutura e linguagem.
- Observe o uso de cores e imagens para criar uma experiencia infantil mais atrativa.
- Nao edite os arquivos originais desta pasta.
- Ao criar uma nova apresentacao, salve uma nova copia fora de `referencias/`.
- Se uma referencia tiver texto menor que 24 pt, corrija esse ponto na nova apresentacao.
